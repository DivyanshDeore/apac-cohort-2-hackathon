# AI-Powered Decision Intelligence Platform

A single-file Streamlit prototype that helps communities, organizations, and city stakeholders analyze civic data, detect patterns, forecast outcomes, generate recommendations, and make better decisions.

## Features

- Community decision readiness dashboard
- Natural-language question answering
- Pattern and anomaly detection
- Predictive forecasting
- Citizen feedback intelligence
- AI-style recommendations
- Automation-ready workflow planning
- Google Cloud deployment mapping
- No external APIs
- No LLM calls
- Lightweight and GitHub-friendly

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
